  • What is Javascript?

    Java Script which is popularly known as JS , is one of the most popular language of programming. It is the programming language mainly used for designing web base applications. The main plus point about JavaScript is that it can not only run in the browser but also on the server and on every platform which has JavaScript engine. And the best thing about it is that it supports almost majority of bbrowsers like chrome, safari, IE . Many big companies like Google, microsoft, apple are using javascript for thyeir web base develpoment. It is easy to learn as compared top other programming languages and also it is user friendly.
